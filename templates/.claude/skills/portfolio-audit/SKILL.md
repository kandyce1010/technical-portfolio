description: Audit every page for stale content, broken links, nav inconsistencies, and missing alt text. Reports findings without making changes.
---

Review every page of the portfolio and report what needs updating. Don't make changes — just report.

Read every .html file in the current directory.

Check for:
1. **Stale content** — dates, roles, company names, event banners referencing past events
2. **Broken or placeholder links** — `href="#"`, `href="URL"`, obviously wrong URLs
3. **Inconsistencies** — nav order differs between pages, active state wrong, footer year off
4. **Missing alt text** — images without descriptive alt attributes
5. **Content gaps** — sections that look like placeholders or are clearly incomplete

Format the report as a checklist grouped by file. Flag high-priority items (stale role/company info, broken nav) separately at the top.
