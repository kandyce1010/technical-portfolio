Add a new blog post entry to blog.html.

Ask me for:
- Post title
- Publication date (or "draft" if not published yet)
- Category (e.g. MLOps, AI/ML, Serverless, Developer Tools, Career)
- 2-3 sentence excerpt
- Tags (3-5 keywords)
- URL (or leave blank if draft)

Add the card at the top of the blog grid (most recent first).
For drafts, use `#` as the href and add a `(Coming Soon)` note after the title.

Match the existing `.blog-card` structure in the file.
