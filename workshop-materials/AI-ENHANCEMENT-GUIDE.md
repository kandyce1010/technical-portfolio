# AI Enhancement Guide - Phase 2 of Workshop

## Overview
This guide is for **Phase 2** of the workshop (minutes 50-75), after participants have built their core 3-page portfolio manually. Use these prompts to enhance your content with AI coding assistants.

**Compatible with:**
- Amazon Q Developer
- Claude Code (recommended)
- GitHub Copilot
- Cursor
- Other AI coding assistants

---

## When to Use This Guide

✅ **Use this guide if:**
- You have an AI coding assistant installed and working
- You've completed Phase 1 (3-page portfolio with your content)
- You want to improve your content quality and polish

❌ **Don't use this guide if:**
- You haven't finished your 3 core pages yet
- Your AI tool isn't working (use manual polish instead)
- You prefer to refine content manually

---

## Quick Start Prompts

### 1. Improve Project Descriptions
**Use when:** Your project descriptions feel bland or too technical

**Prompt:**
```
Rewrite this project description to be more engaging and highlight 
the technical challenges I solved. Make it compelling for recruiters:

[paste your current project description]

Keep it to 2-3 sentences and maintain a professional tone.
```

### 2. Generate Professional Summary
**Use when:** Your about page needs a stronger opening

**Prompt:**
```
Based on this information from my resume, write a compelling 2-3 
sentence professional summary that highlights my unique value:

Background: [paste your career highlights]
Skills: [list your key technical skills]
Goals: [what you're looking for]

Make it engaging but professional.
```

### 3. Create Blog Post Ideas
**Use when:** You want to add content to your blog page

**Prompt:**
```
Suggest 3 blog post topics based on my background in [your field] 
that would showcase my expertise and be valuable to other developers.

For each topic, provide:
- Catchy title
- 2-sentence description
- Why it matters

My background: [brief summary of your experience]
```

### 4. Enhance About Page Content
**Use when:** Your about page feels too resume-like

**Prompt:**
```
Rewrite this career timeline entry to be more narrative and 
engaging while keeping the key facts:

[paste your current timeline entry]

Make it tell a story about my growth and impact.
```

### 5. Polish Contact Page
**Use when:** Your contact page needs better copy

**Prompt:**
```
Write a friendly but professional introduction for my contact page 
that encourages people to reach out. 

My focus areas: [what you want to be contacted about]
My personality: [professional, friendly, collaborative, etc.]

Keep it to 2-3 sentences.
```

### Improve Technical Writing
```
Make this technical description more accessible to non-technical 
recruiters while keeping the technical details:

[paste your content]
```

### Add Personality
```
This content is accurate but feels too formal. Rewrite it to show 
more of my personality while staying professional:

[paste your content]

```

### SEO Optimization
```
Suggest 5-7 keywords I should naturally incorporate into my 
portfolio based on this content:

Role I'm targeting: [job title]
My skills: [list skills]
My experience: [brief summary]
```

### Create Project Case Study
```
Expand this project description into a mini case study format:

Current description: [paste description]

Include:
- Problem statement
- My approach
- Technologies used
- Results/impact
- Key learnings

Keep it concise (4-5 sentences total).
```

## Tips for Effective AI Enhancement

### Do's ✅
- **Start with your content:** AI enhances, doesn't create from scratch
- **Be specific:** Give context about your audience and goals
- **Iterate:** If first result isn't perfect, refine your prompt
- **Keep your voice:** Edit AI output to sound like you
- **Test readability:** Read it out loud - does it sound natural?

### Don'ts ❌
- **Don't use AI-generated content verbatim:** Always personalize
- **Don't lose authenticity:** Your unique experiences matter
- **Don't over-polish:** Some personality is good
- **Don't forget to fact-check:** AI can be wrong about technical details
- **Don't rely on AI for everything:** Manual polish is valuable too

## Workflow for Phase 2 (15 minutes)

### Minutes 50-55: Quick Wins
1. Pick your weakest project description
2. Use "Improve Project Descriptions" prompt
3. Copy enhanced version to your portfolio
4. Test in browser

### Minutes 55-60: About Page Polish
1. Use "Generate Professional Summary" prompt
2. Replace or enhance your current summary
3. Test in browser

### Minutes 60-65: Add Blog Content
1. Use "Create Blog Post Ideas" prompt
2. Add 2-3 blog post placeholders to blog.html
3. Test in browser

### Minutes 65-70: Final Polish
1. Review all pages
2. Use AI to improve any weak spots
3. Ensure everything flows well

### Minutes 70-75: Test & Prepare for Demo
1. Click through all pages
2. Test on mobile view (browser dev tools)
3. Fix any issues
4. Ready to show!

## Using Different AI Tools

### Claude Code
- Install: `npm install -g @anthropic-ai/claude-code`, then run `claude` in your portfolio directory
- Use the `/enhance-content` skill for guided content improvement prompts
- Ask Claude directly in chat for any one-off improvements
- References your actual files — give it the page name and ask for specific changes
- Great for: rewriting project descriptions, improving your About narrative, generating blog ideas

### Amazon Q Developer
- Use inline suggestions as you type
- Use chat for longer prompts
- Can reference your entire codebase

### GitHub Copilot
- Use inline suggestions
- Use Copilot Chat for prompts
- Works in VS Code, JetBrains IDEs

### Cursor
- Use Cmd+K (Mac) or Ctrl+K (Windows) for inline edits
- Use chat for longer prompts
- Can reference multiple files

### Other AI Assistants
- Copy prompts to your AI tool's chat
- Paste results back into your code
- Iterate as needed

## Troubleshooting

**AI suggestions don't match my style:**
- Add "Write in a [casual/formal/technical] tone" to your prompt
- Edit the output to match your voice

**AI is too verbose:**
- Add "Keep it concise, maximum 2-3 sentences" to your prompt
- Manually trim the output

**AI doesn't understand context:**
- Provide more background in your prompt
- Break complex requests into smaller prompts

**AI tool isn't working:**
- Switch to manual polish (Track B)
- You already have a working portfolio!

---

## Remember

**The goal of Phase 2 is enhancement, not perfection.**

You already have a working portfolio from Phase 1. AI helps you polish and improve, but your content and authentic voice are what matter most.

If AI isn't helping or is taking too long, switch to manual polish - both tracks lead to success!
